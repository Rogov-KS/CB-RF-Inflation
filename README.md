# CB-RF-Inflation

Модель воспроизводит результаты из [аналитической записки Банка России по инфляции](https://www.cbr.ru/Content/Document/File/187425/analytic_note_20260310_dip.pdf), ключевой ставке и потерям темпов роста ВВП в нескольких сценариях денежно-кредитной политики.

## Структура

- `params.py` — численные параметры модели.
- `policies.py` — стратегии расчёта ключевой ставки.
- `simulation.py` — вычислительное ядро симуляции.
- `plots.py` — все функции построения графиков.
- `main.py` — воспроизведение всех результатов в скриптовом формате.
- `notebooks/reproduce.ipynb` — воспроизведение с пояснениями в ноутбуке.

## Как запустить

1) Установить зависимости:

```bash
python3 -m pip install .
```

2) Запустить скрипт со всеми графиками:

```bash
python3 main.py
```

3) Или открыть ноутбук:

- `notebooks/reproduce.ipynb`
- последовательно выполнить ячейки сверху вниз.
