# CB-RF-Inflation

В репозитории написан код моделирования, описанный в [аналитической записки Банка России](https://www.cbr.ru/Content/Document/File/187425/analytic_note_20260310_dip.pdf) по инфляции, ключевой ставке и потерям темпов роста ВВП в нескольких сценариях денежно-кредитной политики.

А также исследованы другие стратегии, не упомянутые в записке ЦБ РФ

## Ноутбуки с исследованиями

- [`notebooks/reproduce.ipynb`](https://github.com/Rogov-KS/CB-RF-Inflation/blob/master/notebooks/reproduce.ipynb) — полное воспроизведение графиков из аналитической записки с пошаговыми пояснениями в Markdown.

- [`notebooks/new_strategies.ipynb`](https://github.com/Rogov-KS/CB-RF-Inflation/blob/master/notebooks/new_strategies.ipynb) — исследование альтернативных режимов ключевой ставки (константные 9/11/13/16/20/30% и двухпериодная стратегия `13% -> 11%`).

## Исследованные стратегии

### Стратегии из аналитической записки

- **Сценарий 1**: ключевая ставка фиксирована на уровне `3%` на всём горизонте.
- **Сценарий 2**: ключевая ставка равна текущей инфляции (`i_t = pi_t`).
- **Сценарий 3**: в первые `4` квартала ставка следует сценарию ошибки, затем включается правило Тейлора.
- **Сценарий 4**: в первые `12` кварталов ставка следует сценарию ошибки, затем включается правило Тейлора.

### Новые стратегии

- **Сценарий 5**: константная ставка `rate` на всём горизонте (в ноутбуке рассмотрены `9%`, `11%`, `13%`, `16%`, `20%`, `30%`).
- **Двухпериодная стратегия**: первые `2` года ставка `13%`, далее ставка `11%`.
- Дополнительные сценарии собраны в ноутбуке [`notebooks/new_strategies.ipynb`](https://github.com/Rogov-KS/CB-RF-Inflation/blob/master/notebooks/new_strategies.ipynb).

## Замечания по исследованным стратегиям

- В исследовании стратегий, которых не было в аналитической записке, получились моделирования, в которых ВВП обгоняет потенциальный ВВП. А это говорит о неприменимости модели (возможно из-за нарушения ограничений модели, возможно из-за чего-то ещё)

## Структура

- `params.py` — численные параметры модели.
- `policies.py` — стратегии расчёта ключевой ставки.
- `simulation.py` — вычислительное ядро симуляции.
- `plots.py` — все функции построения графиков.
- `main.py` — воспроизведение всех результатов в скриптовом формате.
- `notebooks/reproduce.ipynb` — воспроизведение с пояснениями в ноутбуке.
- `notebooks/new_strategies.ipynb` — эксперименты с новыми стратегиями ставок (константные и двухпериодные режимы).
